Iframe Utils Example

Auto-initialization Example

This iframe will automatically resize based on its content:

Code:

<iframe 
  src="path/to/app/index.html"
  data-iframe-resize='{"minHeight": 400, "maxHeight": 2000}'
  frameborder="0"
  scrolling="no">
</iframe>

Manual Initialization Example

This iframe uses manual initialization:

Code:

<iframe id="manual-iframe" src="path/to/app/index.html"></iframe>

<script>
  const resizer = new IframeResizer({
    minHeight: 400,
    maxHeight: 2000
  });
  resizer.init(document.getElementById('manual-iframe'));
</script>
